PHP - beginner

  1. install PHP + Apache (WAMP on Windows) to start coding
  2. visit PHP Manual for "hello world" examples and full documentation
  3. learn it all from any PHP tutorial, until you learn it all:
    • syntax, variables, conditional instructions, loops
    • arrays, functions
    • objects and classes
  4. learn about the web:
    cookies, session, HTTP variables, GET and POST requests
  5. play with database: install MySQL and use PDO to read and add some records
