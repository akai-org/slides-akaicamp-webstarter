PHP - intermediate

  1. install WordPress and tweak it - make your own theme, create a widget, customize some plugins
  2. learn the MVC way to create modern web applications
  3. create a web application in modern PHP framework:
    • Symfony 2
    • Laravel
  4. Beware of old, unused frameworks / libraries!

    Use only those libraries, that are still maintained and being updated! (just check the last commit date on GitHub)

  5. learn about testing, security
  6. don't stick to only-PHP - learn also other languages =)
